# Cenfomon2023

## Descripción

Este proyecto es un juego basado en "Pokémon", utilizando de base tutoriales del canal Game Dev Experiments en Youtube.

## Características

Sus funciones base son: 

- **Comprar objetos
- **Interactuar con npcs
- **Pelear contra entrenadores y cenfomones salvajes

## Conclusión

Este proyecto fundamentalmente fue hecho para poner a prueba los patrones de diseño.
